# CreNexus Console Static Export

This directory contains a packaged static export of the CreNexus Creative Console.

- `index.html` is the exported console page.
- `support.js` is the runtime required by the exported page.
- `thumbnail.webp` is the bundled preview thumbnail from the export package.

Open `index.html` directly in a browser to preview the static version. The main React/Vite source remains in `examples/starbridge_frontend/src`.
