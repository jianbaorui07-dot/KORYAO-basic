## 本地整理位置

新下载的源码项目、安装包和调研资料先放在本机下载收件箱，路径只保存在本地环境变量：

```text
STARBRIDGE_DOWNLOAD_INBOX
```

建议源码整理位置：

```text
%STARBRIDGE_DOWNLOAD_INBOX%\codex\GitHub项目源码_保持可运行
```

## 不发布内容

以下内容只保留本机，不进入 GitHub：

- 历史网页 demo、虚拟宠物 demo、PPT 工作区。
- 报告生成脚本、图片素材、样式参考文档。
- 输出目录、缓存目录、日志和临时文件。
- 模型文件、生成图片、私有 `.blend`、客户 DWG、PSD/AI 工程、剪映草稿、导出视频、Photoshop/Illustrator 安装路径和本机素材路径。
- 账号、密码、验证码、Cookie、Token、OAuth 缓存、支付信息和浏览器资料。
