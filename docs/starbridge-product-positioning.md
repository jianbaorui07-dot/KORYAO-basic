# StarBridge 产品定位

StarBridge 的定位是让 Codex 以公开安全、可检查的方式接入本地创意软件。

## 当前定位

- 公开仓库保存协议、状态检查、只读探针、最小示例和中文文档。
- 本机商业软件、模型、素材、客户项目和真实输出留在用户机器上。
- bridge 先提供状态与安全边界，再逐步扩展到受控动作。

## 当前 bridge

- `comfyui`：状态检查、workflow 验证、summary、dry-run queue。
- `jianying_capcut`：状态检查、timeline spec、safe `draft_plan` 生成。
- `photoshop`、`illustrator`、`blender`、`autocad`：以状态和探针为主，动作范围保持保守。

## 不承诺

- 不托管模型、素材或生成结果。
- 不代替用户授权商业软件。
- 不自动读取客户工程或真实草稿。
- 不在 CI 中运行本机桌面软件。
