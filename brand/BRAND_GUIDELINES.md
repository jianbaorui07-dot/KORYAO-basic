# StarBridge 品牌指南

## 品牌核心

- 主名称：**StarBridge**
- 中文描述：**本地创意工作台**
- 图形概念：**星桥轨迹**
- 品牌承诺：让创意工具在用户自己的电脑上安全协同，用户文件不上传到 StarBridge 服务器。

禁止使用“AI 万能创作平台”“一键控制所有软件”“全自动替代设计师”“无限智能”等夸大表述。

## 单一源文件

`starbridge-symbol.svg` 是图标几何和颜色的唯一源文件。PNG、ICO、favicon、安装器图标、应用图标和自包含锁定组合均由 `scripts/build-brand-assets.mjs` 生成，不手工编辑导出文件。

其他 SVG 文件的职责：

- `starbridge-wordmark.svg`：精确文字与 Windows 系统字体规则；
- `starbridge-lockup-horizontal.svg`：横向品牌组合；
- `starbridge-lockup-compact.svg`：紧凑组合；
- `starbridge-icon-grid.svg`：安全区、小尺寸与明暗背景检查；
- `starbridge-readme.svg`：GitHub README 自包含品牌图。

这些组合文件均由构建脚本重建；标志路径仍来自 `starbridge-symbol.svg`。

## 颜色

| 角色 | 色值 | 用途 |
| --- | --- | --- |
| 深海军蓝 | `#153A73` | 图标底、导航与稳定品牌识别 |
| 主蓝 | `#2563EB` | 主要操作与交互焦点 |
| 青蓝 | `#0EA5C6` | 第二轨迹、连接状态和强调 |
| 浅青蓝 | `#73D6ED` | 四角星节点和小面积高光 |
| 白 | `#FFFFFF` | 主轨迹与深色表面内容 |

不使用霓虹外发光、金属、玻璃、复杂渐变或星空纹理。

## 尺寸与安全区

- 应用图标最小使用尺寸：16×16 px。
- 单独图形四周至少保留图标画布宽度的 `1/8` 安全区。
- 16–24 px 时不添加文字或额外装饰。
- 横向组合小于 160 px 宽时隐藏中文描述，使用紧凑组合。
- 不改变轨迹粗细比例，不移动四角星节点，不把轨迹裁成独立字母 S。

## 明暗背景

应用图标保留深海军蓝底，因此可同时用于浅色和深色 Windows 任务栏。单色场景由构建流程生成审查预览；不得把青蓝轨迹改成低对比灰色。

## 字体与文字

品牌文字使用 Windows 系统字体栈：`Segoe UI Variable, Segoe UI, Arial, sans-serif`。中文界面使用 `Microsoft YaHei UI` 和 `PingFang SC` 回退。不得把第三方字体文件提交到仓库，也不得把字母轮廓伪装成原创字体资产。

## 构建

```powershell
npm.cmd run brand:build
```

输出包含 16、24、32、48、64、128、256、512 px PNG、多尺寸 Windows ICO、favicon、Tauri 应用图标、README 品牌图和安装器图标。构建不访问网络。
